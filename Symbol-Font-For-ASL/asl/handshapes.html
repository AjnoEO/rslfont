ASL Handshapes

This page provides a side-by-side comparison of handshape representations in both ASCII-based and symbol-based writing systems for ASL.

Handshape is only one of the five components that make up any ASL sign.

Some systems group similar handshapes together and represent them as a single symbol for readability and simplicity. Other systems try to provide a way to represent small variations in handshape with precision and detail.

This page lists the 51 basic handshapes included in Symbol Font For ASL, and the equivalent symbol or code for that handshape under each writing system. If a writing system does not account for the handshape, I have listed either the most similar handshape available, or an attempt to approximate the handshape using that system's rules for representing arbitrary handshapes (where possible).
