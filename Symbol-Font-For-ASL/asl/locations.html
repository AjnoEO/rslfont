ASL Locations

This page provides a side-by-side comparison of location representations in both ASCII-based and symbol-based writing systems for ASL.

  • Location is only one of the five components that make up any ASL sign.

The tables below compare the different symbols used to represent location in several writing systems. The leftmost column shows the symbol used in Symbol Font For ASL.
