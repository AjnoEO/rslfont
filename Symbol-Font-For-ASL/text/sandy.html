Below is an excerpt from one of New York City Mayor Bloomberg's emergency announcements during Hurricane Katrina, which were interpreted live in ASL by Lydia Callis.

Each paragraph is preceded by a transcription of its ASL interpretation, as transcribed from the video. The transcription system used is called Symbol Font For ASL.

As with any writing system, long sequences of symbols will initially seem unreadable, and it can be a challenge to work with the system long enough to become comfortable with the symbols' visual appearance.

As you read, consider whether the writing system provides enough information so that the original meaning is not lost. For ASL, this is an especially important concern, since so much information is communicated by body language and facial expression.
